Module: resource_objects

Classes

AttributeObject

Represents a Python data attribute and has its name and value.

BaseObject

A base class for objects - modules/classes/functions. It stores the object and its name, docstring and source code.

ClassObject

Represents a Python class and has its methods and attributes.

FunctionObject

Represents a Python function/method and has its signature additional to the inherited name, docstring and source code.

ModuleObject

Represents a Python module with all of its submodules, classes, functions and attributes.

Resource

Represents a resouce. Contains a list of ModuleObject modules from the module tree of that resource.

SimpleModuleObject

Represents a module object but with limited attributes. It doesn't recursively initialize its submodules to prevent the creating of giant composite objects.
